# Ai Movie Searcher 🎬
**Умный Telegram-бот для поиска фильмов с помощью ИИ**

Автоматизированный бот для получения подробной информации о фильмах, мультфильмах и сериалах с использованием искусственного интеллекта. Работает на базе новейших нейросетей Google Gemini, поддерживает ротацию API-ключей для обхода лимитов и имеет гибкие настройки для ограничения работы в определенных чатах и ветках.

[🇬🇧 Read in English](https://github.com/newfpv/AiMovieSearcher/blob/main/README.md)

![Demo](https://github.com/newfpv/AiMovieSearcher/blob/main/demo.gif)

## ✨ Возможности
- **Парсинг через ИИ:** Выдает структурированное описание, год выпуска, тип и жанры по одному названию.
- **Умная ротация ключей:** Поддерживает список из нескольких API-ключей. Если ключ или модель достигает лимита, бот автоматически переключается на следующий доступный.
- **Современные модели:** Оптимизировано под последние поколения Gemini (Flash и Flash Lite). 
- **Вайтлист чатов:** Ограничивает ответы бота только разрешенными ID групп и веток (топиков).
- **Docker Ready:** Простой и быстрый запуск через `docker-compose`.

## 🔑 Как получить API-ключ Google
Google предоставляет бесплатные API-ключи для своих нейросетей, с ограничением на количество запросов в минуту.

**Инструкция:**
1. Перейдите в [Google AI Studio](https://aistudio.google.com).
2. Авторизуйтесь под своим Google-аккаунтом.
3. Посмотрите в левый нижний угол экрана и нажмите **"Get API key"**.
4. Сгенерируйте и скопируйте свой ключ. *(Совет: настоятельно рекомендуется создать несколько ключей с разных аккаунтов для обеспечения стабильной работы бота).*

### 📊 Бесплатные лимиты моделей:
- **Gemini 3 Flash:** 20 запросов / день
- **Gemini 2.5 Flash:** 20 запросов / день
- **Gemini 3.1 Flash Lite:** 500 запросов / день
- **Gemini 2.5 Flash Lite:** 20 запросов / день

## 🛠 Кастомизация
Вы можете легко настроить поведение бота, отредактировав исходный код:
- **Замена моделей:** Вы можете заменить стандартные модели на другие. Полный список доступных моделей Google Gemini можно найти [здесь](https://ai.google.dev/gemini-api/docs/models).
- **Настройка промпта:** Измените промпт для ИИ в коде, чтобы поменять язык ответов, настроить форматирование или добавить новые критерии в результаты поиска.

## 🚀 Установка и запуск

**1. Клонирование репозитория**
```bash
git clone [https://github.com/newfpv/AiMovieSearcher.git](https://github.com/newfpv/AiMovieSearcher.git)
cd AiMovieSearcher
```

**2. Настройка переменных**
Откройте файл `docker-compose.yml` и найдите раздел `environment`. Просто замените примеры на свои актуальные данные:


```yaml
    environment:
      - TG_BOT_TOKEN=YOUR_BOTFATHER_TOKEN
      - API_KEYS=KEY_1,KEY_2,KEY_3 # Разделите несколько ключей запятыми
      - GROUP_THREAD_MAP=12345678;-1234567643224:5
```


**Как работает `GROUP_THREAD_MAP`:**

  * `12345678` -\> Работает в личных сообщениях с этим ID пользователя.
  * `-1002` -\> Работает во всей группе `-1002`.
  * `-1001:45` -\> Работает ТОЛЬКО в ветке `45` группы `-1001`.
  * *Разделяйте разные ID точкой с запятой (`;`). Если хотите, чтобы бот отвечал всем, просто оставьте эту переменную пустой.*

**3. Запуск бота**

```bash
docker-compose up -d --build
```

## ⚙️ Решение проблем

  - **Бот отвечает "Sorry, keys are overloaded":** Вы достигли лимитов Google API. Добавьте больше ключей в переменную `API_KEYS` или подождите минуту.
  - **Бот игнорирует сообщения:** Проверьте настройки `GROUP_THREAD_MAP`. Убедитесь, что ID групп и веток указаны верно. Если оставить переменную полностью пустой, бот будет отвечать всем.
  - **Предупреждение Flood control exceeded в логах:** Telegram ограничивает частоту отправки сообщений ботом. Бот автоматически подождет нужное время и повторит попытку; он не упадет с ошибкой.
